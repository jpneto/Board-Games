MU TORERE

Traditional - Ngati tribe - New Zealand

This game is played on a 3x3 board with the following adjacencies:

and initial setup:

 
TURN - On each turn, each player moves a friendly stone to an adjacent empty cell.
Exception: a player can only move a stone to the center, if that stone is adjacent to an enemy stone.
GOAL - Wins the player that stalemates the opponent.
 
An example

White moves and wins by moving a2 to a1. Then Black must reply a3-a2 and is stalemated by b3-a3.

With two good players, this game is a draw. The Zillions package includes this game.
